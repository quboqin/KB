🌐 局域网内访问桥接光猫后台指南
针对“光猫桥接 + 小米路由拨号”的网络环境，以下提供三种访问光猫（192.168.1.1）的解决方案，按难易程度和优雅程度分类：
方案一：一根网线永久打通（🌟 最优雅，随时随地访问）
既然路由层不通，我们就利用物理层和您的 旁路由 (192.168.123.99) 来做一次“网段欺骗”和 NAT 转发。
前置物理准备：
找一根多余的网线，一头插在 光猫的 LAN2（或任意空闲 LAN 口），另一头插在 小米主路由的 LAN 口（或局域网内的任意交换机上）。
💡 原理：这会将光猫的管理接口直接暴露到您的局域网物理广播域中。
OpenWRT 旁路由设置：
为了让局域网内所有设备都不用改 IP 就能直接访问，我们在旁路由上做一个静态映射：
1. 登录旁路由 (192.168.123.99) 后台。
2. 进入 网络 (Network) -> 接口 (Interfaces) -> 点击 添加新接口 (Add new interface)。
• 名称: Modem
• 协议: 静态地址 (Static address)
• 接口/设备: 选择 br-lan（即您旁路由现有的局域网接口，有的叫 eth0）。
3. 提交后，在基本设置中填写：
• IPv4 地址: 192.168.1.254（给旁路由分配一个和光猫同网段的 IP）
• IPv4 子网掩码: 255.255.255.0
• 注意：不要填写网关。
4. 进入 防火墙设置 (Firewall Settings) 选项卡，分配防火墙区域：
• 情况 A：如果有 wan 区域：直接在下拉列表中选择 wan（因为 wan 默认开启了 NAT，光猫会以为是旁路由在访问它，从而顺利回包）。
• 情况 B：如果没有 wan 区域（旁路由常见情况）：
1. 在下拉框（或输入框）中手动输入一个新名称，例如 modem_zone，然后按回车键新建它。
2. 点击底部的 保存并应用，先让接口生效。
3. 接着，进入菜单的 网络 (Network) -> 防火墙 (Firewall)。
4. 在“区域”列表中找到刚刚新建的 modem_zone，点击右侧的 修改 (Edit)。
5. 【关键步骤】：勾选 IP 动态伪装 (Masquerading) （这就是 NAT 的核心）。
6. 检查下方的 端口转发 (Inter-Zone Forwarding)，设置允许从源区域 lan 转发到目标区域 modem_zone。
7. 点击 保存并应用。
5. 完成设置。
🎉 完成！ 现在，只要您的手机或电脑的网关是指向旁路由（.99）的，直接在浏览器输入 http://192.168.1.1，就能瞬间打开光猫后台！
方案二：设备手动改 IP（⚡ 适合临时修改配置）
如果您不想插额外的网线，也不想改路由器配置，只需要偶尔进一次光猫。
操作步骤：
1. 拿着笔记本电脑来到光猫旁边。
2. 用一根网线，将电脑直接连到 光猫的 LAN2 接口。
3. 进入电脑的网络适配器设置，将获取 IP 的方式改为 手动/静态：
• IP 地址: 192.168.1.2
• 子网掩码: 255.255.255.0
• 网关: 192.168.1.1
4. 浏览器输入 http://192.168.1.1 即可访问。
5. 完事后，记得把电脑的 IP 获取方式改回“自动 (DHCP)”，并将网线插回原处。
方案三：手机直连光猫自带 Wi-Fi（📱 最简单，前提是光猫有 Wi-Fi）
现在很多电信天翼网关（光猫）自身就带路由和 Wi-Fi 功能，即使改了桥接，它的 Wi-Fi 依然会发射管理信号。
操作步骤：
1. 走到光猫附近，查看光猫背面的贴纸，找到它自带的 Wi-Fi 名称（通常是 ChinaNet-xxxx）和默认密码。
2. 用手机直接连接这个 Wi-Fi。
3. 连接后，手机通常会自动获取到 192.168.1.x 的 IP。
4. 此时手机虽然不能上网，但可以直接在浏览器输入 http://192.168.1.1 登录光猫后台。
5. ⚠️ 提示：如果不希望光猫的 Wi-Fi 干扰主路由的信号，建议进去配置完后，在后台把光猫的 Wi-Fi 功能关掉。
💡 架构师提醒
如果您选择了 方案一（插一根线到主路由 LAN），请确保您的光猫在桥接模式下 已经关闭了自身的 DHCP 服务（通常桥接模式默认是关闭的）。如果光猫的 DHCP 还在开启状态，可能会给您局域网内的设备错误地分配 192.168.1.x 的 IP 导致断网。
