葡萄酒的品质随着时间推移会慢慢提升。酿酒公司常把酿酒的年份记录下来，追踪不同年份的酒在时光中的品质变化——这就是"Vintage"（年份）这个词最早的使用场景。

信贷业务和葡萄酒的储存，其实有几分神似。我们同样需要追踪不同时期借出去的资金，监控逾期率随时间的变化。

这，就是 **Vintage 分析** 的本质。

## 一笔借款的"生命档案"

在理解 Vintage 之前，先看看我们对一笔借款都记录了什么：

| 字段 | 含义 | 示例 |
|------|------|------|
| 借款金额 | 给用户放款的金额 | 1000 元 |
| 放款时间 | 资金到达客户手中的时间 | 2020-04-01 |
| 到期时间 | 客户应该还款的截止时间 | 2020-05-01 |
| M1+ 时间 | 逾期 30 天及以上的时间点 | 2020-05-31 |
| 还款时间 | 用户实际的还款时间 | 2020-07-25 |

## MOB：给时间贴上刻度

Vintage 分析的核心工具，是"MOB"—— **Month on Book（在账本上记录了几个月）**。

```
放款月 = 基准月（Vintage 基准）
  ↓
第 1 个自然月末 = MOB0
第 2 个自然月末 = MOB1
第 3 个自然月末 = MOB2
……
```

核心公式：

```
逾期率 = 第 k 个 MOB 月，逾期 M1+ 的剩余本金 / 基准月放款总额
```

> 分母是基准月的**总放款额**，永远不变。分子则随着时间推移不断更新——哪笔钱逾期了，算进去；哪笔钱还款了，剔除出去。

## 单笔借款：从理论到计算

假设 2020 年 1 月（基准月），只有一笔 1000 元的借款，该借款在 3 月未还款产生逾期。

**时间线：**

```
1月放款         到期（2月底应还）    逾期发生         用户还款
    |               |                  |                 |
    ●───────────────●──────────────────●─────────────────●
    |               |                  |                 |
  MOB0            MOB1               MOB2              MOB4
  (1月末)         (2月末)             (3月末)           (5月末)
  [未逾期]        [未逾期]            [M1+ 逾期!]      [已还款]
```

**Vintage 表格（30+ 逾期口径）：**

| MOB | 观测时点 | 逾期 M1+ 金额 | 基准月放款额 | 逾期率 |
|-----|---------|--------------|------------|--------|
| MOB0 | 1 月末 | 0 | 1000 | **0%** |
| MOB1 | 2 月末 | 0 | 1000 | **0%** |
| MOB2 | 3 月末 | 1000 | 1000 | **100%** |
| MOB3 | 4 月末 | 1000 | 1000 | **100%** |
| MOB4 | 5 月末 | 0 | 1000 | **0%**（用户还款，清零）|

> **注意：** MOB4 逾期率回到 0，是因为我们记录的是"**当前逾期状态**"的快照。用户还款后，就不再处于 M1+ 状态了。这意味着 Vintage 曲线可以出现下降——它不是历史最高水位，而是当下截面的写真。

## 多笔借款：真实世界更复杂

实际工作中，一个基准月会有大量借款同时放出。

假设 1 月共放出 5 笔 × 1000 元 = 5000 元：

| 借款编号 | 状态说明 |
|---------|---------|
| 借款 1 | MOB2 时逾期，一直未还 |
| 借款 2 | MOB2 时逾期，MOB3 时还清 |
| 借款 3 | MOB2 时正常还款 |
| 借款 4 | MOB2 时正常还款 |
| 借款 5 | MOB2 时正常还款 |

**Vintage 表格：**

```
MOB0: 逾期金额 0     / 5000 → 逾期率 0%
MOB1: 逾期金额 0     / 5000 → 逾期率 0%
MOB2: 逾期金额 2000  / 5000 → 逾期率 40%  ← 借款1、2 均逾期
MOB3: 逾期金额 1000  / 5000 → 逾期率 20%  ← 借款2 还清，只剩借款1
```

逾期率从 MOB2 的 40% 下降到 MOB3 的 20%，并不意味着"风险变好了"——只是有一笔钱还掉了，当前逾期存量减少了。

## 两种观测口径

Vintage 的核心在于"**选定观测时间点**"，通常有两种做法：

### 统一月末观测（Month End）

在每个自然月的最后一天，统一观测所有在贷借款的状态。

- **优点：** 简单直接，所有基准月的数据在同一时间点汇总
- **适用：** 业务发展早期，数据基础薄弱阶段
- **通用方式**，本文所有示例均基于此口径

### 到期日后第一天观测（Cycle End）

每笔借款各自在其到期日后第一天观测，各笔借款的观测日各不相同。

- **优点：** 精准反映单笔借款的还款行为，不受自然月边界影响
- **适用：** 业务成熟期，追求更精细化分析
- 在业务发展中后期可以更精准地统计分析 MOB & Vintage

## Vintage 图：一眼看穿资产健康趋势

把多个基准月的逾期率曲线画在一张图上，就得到了经典的 **Vintage 图**：

```
逾期率
  |
  |          ●──●
  |         /    \
  |     ●──●      ●──●       ← 1 月份（逾期率较高）
  |    /
  |   ●    ●──●              ← 2 月份（增长趋势减缓）
  |        /    \
  |       ●      ●──●        ← 3 月份（进一步改善）
  |                  \
  |                   ●──●   ← 4 月份（最优）
  |
  +──────────────────────────
   MOB0 MOB1 MOB2 MOB3 MOB4
```

**如何读这张图：**

- **横轴：** MOB 值（从放款到观测经过了几个月）
- **纵轴：** 逾期率
- **每一条曲线：** 代表某个基准月的放款资产
- **曲线整体高度越低：** 该月资产质量越好

> 如果最近几个月的曲线，整体"高度"低于较早月份的曲线，说明资产质量在持续改善；反之，则意味着风险正在上升。

## Vintage 的战略价值：预判，而不是事后复盘

Vintage 分析真正的价值在于 **预测**。

贷款放出去后，需要较长时间才能看到完整的还款表现。我们无法等到所有借款"盖棺论定"再做判断。但通过横向对比不同基准月在早期 MOB（如 MOB1、MOB2、MOB3）上的逾期率表现，可以大致估算：

> "这批新放出去的贷款，最终的逾期率大概会落在哪个区间？"

这个判断，足以让我们提前 3-6 个月调整策略——比如收紧授信标准、调整利率定价、加强贷中预警——而不是等到坏账已经发生才亡羊补牢。

---

就像观察一瓶刚入窖的新酒，凭借早期的色泽和气味，老酿酒师已经能预言它 10 年后的品质。

**MOB 与 Vintage，就是信贷风险管理师手中的那把年份标尺。**
