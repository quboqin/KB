"借款 12000，年化利率 10%，12 个月还清。"

这句话，几乎出现在每一款消费信贷产品的宣传页里。看起来清清楚楚，实则暗藏玄机。

同样是"年化 10%"，同样是"借款 12000、借款 12 个月"，不同的**还款方式**下，你实际支付的利息成本，差距可以将近 **2 倍**。

## 五种还款方式全解析

### 1. 随借随还

随时借款，随时还款，还款后额度恢复。典型产品：花呗借呗、银行大额消费贷。

```
总还款利息 = 贷款本金 × 日利率 × 贷款天数
```

**示例：** 借款 12000，借款期限 12 个月，月利率 = 10%/12

```
总还利息 = 12000 × (10%/365) × 365 = 1200 元
```

### 2. 先息后本

每月只付利息，到期一次性归还本金。适合有一段时间后才能回款的个人或企业。

```
每月还款利息 = 贷款本金 × 月利率
总还款利息   = 贷款本金 × 月利率 × 还款月数
```

**示例：**

```
每月应还利息 = 12000 × 10%/12 = 100 元
总还利息    = 12000 × 10% = 1200 元
```

### 3. 等本等息

每月还相同的本金 + 相同的利息。常见于银行卡现金分期、账单分期。

```
每月还款本息 = (贷款本金 / 还款月数) + 贷款本金 × 月利率
总还款利息   = 贷款本金 × 月利率 × 还款月数
```

**示例：**

```
每月应还本金 = 12000 / 12 = 1000 元
每月应还利息 = 12000 × 10%/12 = 100 元（按初始本金计算）
每月应还本息 = 1000 + 100 = 1100 元
总还利息    = 12000 × 10% = 1200 元
```

> **注意：** 等本等息每月利息按**初始本金**计算，而非剩余本金——这正是埋下"利率陷阱"的关键所在。

### 4. 等额本金

每月还相同本金 + 按**剩余本金**计算的利息。每月还款额逐月递减。常见于房贷。

```
每月还款本息 = (贷款本金 / 还款月数) + (贷款本金 - 已归还本金) × 月利率
总还款利息   = 贷款本金 × 月利率 × (还款月数 + 1) / 2
```

**示例：**

```
每月应还本金 = 12000 / 12 = 1000 元
第 1 月利息  = 12000 × 10%/12 = 100 元
第 2 月利息  = (12000-1000) × 10%/12 = 91.67 元
...
第 12 月利息 = (12000-11000) × 10%/12 = 8.33 元
总还利息    = 12000 × 10%/12 × (12+1)/2 = 650 元
```

### 5. 等额本息

每月还款**总额相同**，其中利息逐月递减、本金逐月递增。住房贷款最常用的还款方式。

```
总还款利息 = 每月还款本息 × 还款月数 - 贷款本金
```

**示例：**

```
每月应还本息 ≈ 1055 元（公式推导得出）
总还利息    = 1055 × 12 - 12000 = 660 元
```

### 五种方式汇总

| 还款方式 | 总还利息 |
|---------|---------|
| 随借随还 | 1200 元 |
| 先息后本 | 1200 元 |
| 等本等息 | 1200 元 |
| 等额本金 | **650 元** |
| 等额本息 | **660 元** |

同样是年化 10%，为什么等额本金的总利息只有一半多？这难道说明等额本金利率更低吗？

答案，藏在 APR 和 IRR 的区别里。

## APR vs. IRR：名义利率与真实利率

### APR（名义利率）

APR = Annual Percentage Rate，年化利率，即对外公示的利率。

计算公式简单：

```
APR = 总息费 / 贷款本金
```

### IRR（真实利率）

IRR = Internal Rate of Return，内部收益率，又称**真实利率**。

它的本质是：考虑了**货币的时间价值**后，借款方实际承受的资金成本。

**通俗比喻：** 假设小明借给张三 10000 元，本息共 11000，约定年化 10%，1 年内还清，有两种还法：

- **方案 A：** 每 6 个月还 5500，分两次还清
- **方案 B：** 1 年后一次性还清 11000

所有人都会选方案 A——因为先收回的 5500 还可以再投资半年，整体收益率更高。**从 IRR 计算上，方案 A 的 IRR 也高于方案 B**，即便两者的 APR 相同。

> IRR 的宏观含义：该笔投资每年能承受的货币最大贬值幅度或通货膨胀率。

### 计算方法：用 Excel IRR 函数

以先息后本为例：

```
现金流：
  第 0 个月（放款）: -12000
  第 1 个月（收利息）: +100
  第 2 个月（收利息）: +100
  ...
  第 12 个月（收本金+利息）: +12100

步骤：
  1. 用 Excel IRR() 函数计算实际月利率
  2. 实际月利率 × 12 = IRR（年化真实利率）
```

### 五种还款方式的 IRR 对比

| 还款方式 | 总还利息 | 实际月利率 | IRR（年化） | APR |
|---------|---------|----------|------------|-----|
| 随借随还 | 1200 元 | 0.80% | **9.57%** | 10.00% |
| 先息后本 | 1200 元 | 0.83% | **10.00%** | 10.00% |
| **等本等息** | **1200 元** | **1.50%** | **17.97%** | **10.00%** |
| 等额本金 | 650 元 | 0.83% | **10.00%** | 5.42% |
| 等额本息 | 660 元 | 0.83% | **10.00%** | 5.50% |

**震惊的发现：**

> 等本等息的 APR 是 10%，但 IRR 高达 **17.97%**——几乎是真实成本的**两倍**！

## 等本等息的"利率陷阱"拆解

为什么等本等息的 IRR 会是 APR 的近两倍？

等本等息每月还 1000 元本金。到了第 6 个月，用户只剩 6000 元借款了。但每月的利息，仍然按**初始 12000 元**计算（每月固定 100 元）——就好像你还没有归还过任何本金一样。

等效于：**用越来越少的本金，承担着与最初完全一样的利息绝对额**。随着时间推移，利息相对于剩余本金的比率不断攀升。

而等额本金/等额本息恰好相反——利息始终按**剩余本金**计算，因此 IRR 与 APR 高度一致。

## 等额本金 vs. 等额本息：谁更划算？

两者的 IRR 都是 10%，只是 APR 不同（5.42% vs 5.50%）。

这揭示了一个重要原理：**利息的多少，不由还款方式决定，而由资金实际占用时间和占用金额决定。**

```
利息 = 资金额 × 利率 × 占用时间
```

等额本金之所以总利息更少（650 < 660），是因为它在还款早期归还本金更快，减少了资金占用时间，自然利息也少。这是数学规律的必然，不存在"省到利息"或"银行吃亏"的说法。

## 监管为何要求披露综合融资成本

在实际产品宣传中，存在一种隐性不对等：

- **等额本金/等额本息类房贷：** 往往宣传较高的真实利率 IRR（"利率仅需 5%"）
- **等本等息类消费分期：** 往往宣传更低的名义利率 APR（"月息仅 0.83%"），而 IRR 实际接近 18%

这种"报低实高"的现象，正是中国央行要求金融机构披露**综合融资成本**（对应 IRR）的根本原因。

## 给消费者的一句忠告

> 在选择信贷产品时，**不能只看 APR**。
>
> 比较不同信贷产品，唯一公平的标准是 **IRR（真实利率）**。
>
> 等本等息的宣传 APR 可能是 10%，但 IRR 近 18%；等额本息的宣传 APR 可能只有 5%，但 IRR 也是 10%——两者的真实成本相同。

利率，看似简单，实则不简单。理解 APR 与 IRR 的差异，是每一个和信贷打交道的人的必修课。
